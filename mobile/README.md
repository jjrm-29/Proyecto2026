# Proyecto2026 Mobile

App movil en Expo/React Native para conectarse al mismo Supabase del proyecto web.

## Configuracion

1. Crea `mobile/.env`.
2. Copia los valores publicos de Supabase del `.env` principal:

```env
EXPO_PUBLIC_SUPABASE_URL=valor_de_VITE_SUPABASE_URL
EXPO_PUBLIC_SUPABASE_ANON_KEY=valor_de_VITE_SUPABASE_API_KEY
```

## Comandos

```bash
npm run start
npm run android
npm run ios
npm run web
```
